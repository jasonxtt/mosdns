# v0.3.2

## Summary

This release focuses on mobile usability and legacy parity updates in the Vue dashboard UI.

## Changed

- mobile navigation:
  - top-level menu keeps one row with refresh button included
  - supports horizontal scrolling instead of wrapping into two lines
- system settings:
  - `高级替换规则` table now supports horizontal scrolling on narrow screens
- data management:
  - cache/stat tables now use horizontal scrolling to avoid clipped table headers
  - scheduler form field widths in `刷新分流缓存` are aligned for better mobile layout
- diagnostic capture:
  - `请求列表` and `分析结果` modules support horizontal scrolling on small screens
- list management:
  - restored legacy `/log` hint descriptions for all manual list types
  - status display now uses `共 X 行`

## Config impact

This release does **not** require a config change for existing users.

No incremental config package update is needed for this release.
