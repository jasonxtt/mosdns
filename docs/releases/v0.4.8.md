# v0.4.8

## Summary

This patch release follows `v0.4.7` and finishes the last mobile `/log`
polish items that surfaced during real-device testing.

- keeps the new `/log` mobile top notice as a smaller bubble-style feedback UI
- fixes the bubble positioning so text no longer drifts beyond the right screen edge
- tightens spacing between the compact primary nav and the secondary nav strip

## Config update behavior

This is a binary-only release.

- required config schema remains `2`
- required config package ID remains `main-config-schema-2`
- no new `config_up.zip` is required for this release

## UI

Highlights in this release:

- mobile `/log` top notices are centered correctly on narrow screens
- long success text in the mobile notice bubble remains inside the viewport
- the secondary nav sits closer to the primary nav on phone-width layouts
