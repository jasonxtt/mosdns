# v0.7.1

## Summary

This binary-only maintenance release fixes two WebUI layouts that could hide
critical actions at particular viewport sizes.

- the upstream editor is mounted at the page root and uses a bounded,
  independently scrollable body, keeping its close and save controls visible
- the rule-management content column can shrink within the page grid, so the
  rightmost save action remains in the viewport without a horizontal scroll
- the "保存全部改动" action stays on one line

## Compatibility boundary

`v0.7.1` keeps config schema `3` and package ID `main-config-schema-3`.
No `config_up` or `config_all` package is required.
