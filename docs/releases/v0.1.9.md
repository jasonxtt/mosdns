# v0.1.9

## Summary

This release continues refining the merged dashboard UI so the main operational focus stays on upstream management and query tooling, while reducing noisy secondary controls.

## Added

- a dedicated `数据管理` section for cache and generated-domain maintenance
- a one-click `清空所有缓存` action in cache management
- an inline confirmation popover for switching between `兼容模式` and `安全模式`

## Changed

- the top-level dashboard layout is further refined so `上游设置` and `系统设置` are easier to scan
- the upstream page header is simplified so the upstream DNS table remains the visual focus
- special-group controls are now presented as a lighter inline management strip instead of a heavy side card
- `重启 MosDNS` is now placed inside the system information module
- `SOCKS5 / ECS IP` is now managed from the system settings area
- dashboard update checks no longer run automatically in the background
- the manual update action is now labeled `检查更新`

## Config impact

This release does **not** require a config change for existing users.

No incremental config package update is needed for this release.

## Notes

- `/` remains the primary merged dashboard entry
- `/legacy`, `/log`, and `/graphic` are still kept for compatibility during transition
