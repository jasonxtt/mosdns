# v0.3.5

## Summary

This release introduces and stabilizes a dynamic real-time `查询趋势` experience in the Vue dashboard, with synchronized KPI updates and smoother live monitoring behavior.

## Added

- dynamic real-time trend panel in overview:
  - polling-based live updates
  - sliding time-window rendering for `请求数` and `平均处理时间`
  - synchronized KPI values: `总查询数 / 平均处理时间 / 当前请求数 / 当前处理时间`
- interactive series controls for toggling trend lines without reinitializing the chart

## Changed

- refined related Vue UI workflows and visual polish across:
  - rules/list/upstream/system operation controls
  - mobile compact layout and module overflow behavior
  - panel background and appearance controls (transparency / glass effect)

## Config impact

This release does **not** require a config change for existing users.

No incremental config package update is needed for this release.
