# v0.3.0

## Summary

This release promotes the Vue-based WebUI to the default root dashboard path and keeps the previous dashboard available under `/log` for compatibility.

## Added

- Vue WebUI is now the primary experience at `/`
- added `经典绿` appearance preset in Vue UI theme options

## Changed

- route mapping swap:
  - `/` serves Vue UI
  - `/log` serves the previous legacy dashboard
- continued Vue UI migration and behavior alignment across:
  - overview
  - query logs and diagnostics
  - rules management
  - upstream management
  - data management
  - system settings
- overview polish:
  - top summary focuses on `总查询数` and `平均耗时`
  - nested detail dialog layering fixed
  - module scrollbars are lighter and less intrusive

## Config impact

This release does **not** require a config change for existing users.

No incremental config package update is needed for this release.
