# v0.3.10

## Summary

This release focuses on Vue WebUI appearance and layout consistency.

It adds an inline solid-color picker for panel background, stores panel solid colors separately for light/dark themes, and reduces extra stacked wrapper panels across several pages for a cleaner structure.

## Added

- inline panel-background solid-color picker in `系统设置 -> 主题与外观 -> 面板背景`:
  - integrated into the same row as URL/upload controls
  - positioned before `上传`
- theme-isolated panel solid-color persistence:
  - `light_color` and `dark_color` are now stored independently
  - legacy `color` remains supported for backward compatibility

## Changed

- simplified page chrome/subnav layering:
  - removed redundant top wrapper panels on overview/data/upstream/system pages
  - query/rules subnav switched to strip-style layout with divider
- tightened layout and alignment in list/data/system modules to reduce duplicate panel backgrounds and spacing noise
- panel-background preview now reacts immediately after theme switch
- reduced iOS Safari overlap risk with safer mobile CSS behavior
- updated embedded Vue asset query version for reliable browser cache refresh

## Config impact

This release does **not** require manual `config_up.zip` updates.

Existing panel background `color` configuration remains usable and is normalized to theme-isolated fields on subsequent save.
