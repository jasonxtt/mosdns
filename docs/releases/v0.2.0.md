# v0.2.0

## Summary

This release promotes the Vue-based WebUI at `/log` to the main maintained dashboard path for ongoing feature work, while keeping behavior aligned with the legacy UI workflow.

## Added

- Vue 3 modular UI implementation for:
  - overview
  - query logs and diagnostics
  - rules management
  - upstream management
  - data management
  - system settings
- modal-based editing and detail dialogs across migrated modules
- domain-set ranking donut chart plus percentage display in overview
- top-domain row click detail view in overview

## Changed

- `/log` navigation structure and key interactions were aligned with legacy dashboard behavior for smoother migration testing
- upstream management removed redundant local refresh action and follows global refresh behavior
- overview ranking tables were adjusted to adaptive layouts to reduce horizontal dragging on normal desktop widths
- system mode description placement and wording were updated:
  - `兼容模式`: 表外域名优先国内dns解析，保证速度
  - `安全模式`: 表外域名仅用国外dns解析，阻止dns泄漏

## Config impact

This release does **not** require a config change for existing users.

No incremental config package update is needed for this release.
