# v0.1.8

## Summary

This release merges the previous split WebUI entry points into a new dashboard at `/` and retires the old `config_tom.zip` template reference.

## Added

- a new merged dashboard UI at `/`
- top-level navigation for:
  - `概览`
  - `查询日志`
  - `规则管理`
  - `上游`
  - `系统`
- the legacy root page is still available at `/legacy` for transition

## Changed

- the old `:9099` log-capture and analysis workflow is now integrated into `查询日志`
- upstream management is promoted to a top-level area instead of being buried under system settings
- the dashboard layout now separates query, rule, upstream, and system functions more clearly
- README now points to the maintained config packages:
  - `mosdns/config/config_all.zip`
  - `mosdns/config/config_up.zip`
- the old `config_tom.zip` template is retired

## Config impact

This release does **not** require a config change for existing users.

For config downloads going forward:

- use `mosdns/config/config_all.zip` for fresh deployments or full template replacement
- use `mosdns/config/config_up.zip` for incremental config updates

## Notes

- `/log` and `/graphic` are still kept for compatibility during the transition
- `/legacy` keeps the previous root UI available while the new dashboard is being finalized
