# v0.3.3

## Summary

This release further refines mobile layout behavior to keep panel containers within viewport width and only scroll where needed.

## Changed

- top-level mobile menu:
  - keeps one line including refresh button
  - removed horizontal scrolling for the menu itself
  - reduced button spacing/font sizing for narrow screens
- system settings:
  - `高级替换规则` table scrolls inside its table area on small screens
- data management:
  - cache-management table is compacted on mobile first
  - horizontal scroll appears only when columns still cannot fit
  - `域名列表统计` and `刷新分流缓存` modules stay within screen width
- shared layout robustness:
  - panel/grid items are constrained to prevent module-level overflow beyond viewport

## Config impact

This release does **not** require a config change for existing users.

No incremental config package update is needed for this release.
