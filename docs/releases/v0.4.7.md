# v0.4.7

## Summary

This release finishes a round of operator-facing WebUI work focused on runtime
controls, the compatibility `/log` UI, and mobile usability.

- system settings now expose runtime domain-generation controls without
  requiring external config package changes
- the legacy `/log` UI now ships as its own embedded `vue-log1` frontend build
- `/log` overview and system-settings flows were reorganized for easier daily
  operation
- mobile `/log` navigation was rebuilt to avoid horizontal overflow, drifting
  subnav placement, and clipped square background blocks

## Config update behavior

This is a binary-only release.

- required config schema remains `2`
- required config package ID remains `main-config-schema-2`
- no new `config_up.zip` is required for this release

## UI

Highlights in this release:

- runtime `域名表自生成` switches persist in `/cus/mosdns/webinfo`
- `/log` overview now shows grouped upstream DNS statistics for enabled
  upstreams
- `/log` system settings use flatter maintenance / behavior / appearance /
  log-refresh sections
- `/log` appearance settings use a full-width two-card layout
- mobile `/log` now uses:
  - a compact single-row `MosDNS 仪表盘` header
  - a single-line compact primary nav
  - a stable rounded secondary nav strip
  - no horizontal page overflow on iPhone-width layouts
