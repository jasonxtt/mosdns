# v0.3.8

## Summary

This release improves Vue dashboard appearance controls by adding reusable background-image history, per-theme text-color customization, and clearer selected-state visibility for second-level tabs.

## Added

- panel background history management:
  - list historical uploaded images
  - select historical image for immediate reuse
  - delete single history item
  - clear all history items
- per-theme text color settings:
  - light/dark text color settings are stored separately
  - picker updates are applied and persisted immediately
  - eyedropper support is exposed in supported browsers

## Changed

- second-level tab selected state now has clearer border emphasis for:
  - `实时查询 / 诊断抓取`
  - `本地规则 / 订阅规则 / 拦截规则`
- appearance reset is now a full reset with confirmation:
  - reset theme to `明亮`
  - clear panel background
  - reset transparency to `100%`
  - reset blur to `0px`
  - reset text color to defaults
- removed more hardcoded green text colors from button/text surfaces so text rendering follows active theme text color variables more consistently

## Config impact

This release does **not** require a config change for existing users.

No incremental config package update is needed for this release.
