# v0.6.3

## Summary

`v0.6.3` refines overview statistics behavior and finishes a small round of maintained `/` UI cleanup.

- overview `重置统计` now confirms and restarts `mosdns` instead of relying on browser-local baselines
- overview upstream DNS statistics now always show the backend's real cumulative counters
- mobile/desktop top navigation spacing was tightened, and the extra appearance `取色` buttons were removed

## Upgrade Notes

- this is a binary-only release
- no `config_up` / `config_all` update is required
- existing deployments can update only the binary
