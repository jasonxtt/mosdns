# v0.1.12

## Summary

This release continues refining the merged dashboard by improving rule and upstream list management, adding sortable table headers, and making newly added items appear first by default.

## Added

- added sortable headers for subscription-rule and ad-block rule lists
- added sortable headers for the upstream table
- added a quick toggle in the upstream table header to hide disabled upstreams

## Changed

- newly added subscription rules, ad-block rules, and upstreams now appear first by default until the user applies another sort order
- local rule tabs are now labeled `本地规则 / 订阅规则 / 广告拦截`
- local rule hints were clarified for whitelist, greylist, DDNS, and special-group lists
- the diversion-rule creation dialog now auto-detects the rule name and local `srs/...` path from the rule URL
- saving a diversion subscription now closes the dialog immediately while background download and refresh continue
- README wording now reflects the WebUI module refactor and the current released version

## Config impact

This release does **not** require a config change for existing users.

No incremental config package update is needed for this release.

## Notes

- this release is a source-side WebUI and dashboard polish release
- the merged dashboard remains the primary UI entrypoint at `/`
