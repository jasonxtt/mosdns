# v0.3.1

## Summary

This release introduces a dynamic real-time `查询趋势` module in the Vue dashboard overview.

## Added

- real-time trend panel in overview with polling-based live updates
- sliding time-window trend visualization for query count and average processing time
- synchronized KPI updates:
  - `总查询数`
  - `平均处理时间`
  - `当前请求数`
  - `当前处理时间`
- interactive series toggles for `请求数` and `平均处理时间`

## Changed

- refined trend card layout, compactness, and responsive behavior on narrow screens
- polished monitoring-card interactions and visual consistency

## Config impact

This release does **not** require a config change for existing users.

No incremental config package update is needed for this release.
