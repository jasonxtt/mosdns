# v0.3.11

## Summary

This release fixes dark-theme readability regressions and mobile overlap issues in the Vue WebUI.

It specifically addresses invisible text in `专属分流组` under custom/panel background mode, and fixes narrow-screen overlap in the data-management requery scheduler area.

## Changed

- fixed dark-theme/custom-background text contrast:
  - `专属分流组` name chips are now readable again
  - `msg.success`, `result-badge.fail`, and requery status chips keep visible foreground colors in dark glass backgrounds
- fixed mobile layout overlap in `数据管理 -> 刷新分流缓存`:
  - removed rigid fill-height constraints from inline modules
  - status/action/scheduler rows now wrap safely on narrow screens
- bumped embedded Vue asset query version in `log.html` to force browser cache refresh

## Config impact

This release does **not** require manual `config_up.zip` updates.
