# v0.5.0

## Summary

`v0.5.0` focuses on the dedicated-routing (`special_groups`) workflow.

- dedicated groups can now listen on their own custom DNS port
- each dedicated group can choose whether it still participates in the normal
  port-`53` routing chain
- the maintained `/` UI, the compatibility `/log` UI, and rules management now
  expose the same dedicated-group editing model

## Behavior

- if a dedicated group has no custom port, it continues to work only through
  the normal port-`53` chain
- if a dedicated group has a custom port and `仅自定义端口生效` is not checked,
  both port `53` and the custom port are active
- if a dedicated group has a custom port and `仅自定义端口生效` is checked, that
  group is removed from the port-`53` chain and remains active only on the
  custom listener

## Upgrade Notes

- this is a binary-only release
- no `config_up` / `config_all` update is required
- if the browser still shows stale embedded assets after upgrade, refresh once
